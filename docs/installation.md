Napari-DNA-Damage requires an active installation of both pip and Python to be installed. Due to the TensorFlow dependency of the models utilized in this package, the Python version must be >=3.9 and <3.13.
Then, Napari-DNA-Damage can be installed as follows:
```shell
pip install git+https://github.com/JvB22/napari-dna-damage
```
Next, launch Napari from the command line using:
```shell
napari
```
The Napari graphical user interface should open. To select the DNA-Damage plugin, navigate to 'Plugins' and select 'DNA Damage Workflow'. The DNA Damage widget should appear on the right-hand side of the screen.
