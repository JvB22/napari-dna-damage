# Napari-DNA-Damage
Napari plugin for performing DNA-Damage analysis on microscopy images. Combines automatic segmentation using CellPose or StarDist with spot detection using Spotiflow.<br>
For usage and installation, please check out the [documentation](https://jvb22.github.io/napari-dna-damage/)
